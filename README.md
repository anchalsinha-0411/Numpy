# Numpy

## Description

This Jupyter notebook provides examples and solutions related to NumPy operations, including array creation, manipulation, arithmetic operations, and indexing. It serves as a practical guide for understanding fundamental NumPy concepts demonstrated through various code cells.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Key Concepts Covered](#key-concepts-covered)

## Installation

This notebook can be run directly in Google Colab. No specific installation is required beyond a standard Colab environment. If running locally, ensure you have Python 3 and the following libraries installed:

```bash
pip install numpy
```

## Usage

1.  **Open in Google Colab**: Click the "Open in Colab" badge (if added) or upload the `.ipynb` file to Google Colab.
2.  **Run Cells**: Execute the code cells sequentially to see the demonstrations of NumPy functionalities.
3.  **Assignments**: Complete the assignments provided at the end of the notebook to practice the concepts.

## Key Concepts Covered

- Array creation (`np.array`, `np.arange`, `np.linspace`, `np.ones`, `np.zeros`, `np.eye`, `np.diag`)
- Array reshaping (`.reshape`, `.resize`)
- Array concatenation (`np.vstack`, `np.hstack`)
- Basic arithmetic operations (`+`, `*`, `**`, `//`, `.dot`)
- Array attributes (`.shape`, `.dtype`, `.T`)
- Indexing and slicing in 1D and 2D arrays
- Boolean indexing
- Iterating over arrays

## Assignments

This notebook includes practical assignments to test your understanding:

1.  Create a 3x3 NumPy array of all `True` values.
2.  Replace all odd numbers in a given array with -1.
3.  Find common items between two arrays.
4.  Get the positions where elements of two arrays match.
5.  Swap columns 1 and 2 in a 2D array.
6.  Swap rows 1 and 2 in a 2D array.
